# Intelligent Personal Safety Agent

The Intelligent Personal Safety Agent is a simple AI-based system designed to assist users during emergency situations by detecting potential risks and sending alerts to emergency contacts.

## Features
- Emergency alert system
- Safety monitoring
- Location detection
- Notification to emergency contacts

## Technologies Used
- Python
- Machine Learning
- OpenCV
- NumPy
- Pandas

## Project Structure

main.py – Main program file  
utils.py – Helper functions  
requirements.txt – Required libraries

## How to Run

Install required libraries:

pip install -r requirements.txt

Run the program:

python main.py
